iOS 捷径配置

  1. 获取剪贴板
  2. URL 编码 对剪贴板内容编码
  3. 文本 拼接：http://你的IP:9000/parse?url= + 编码结果
  4. 获取 URL 内容 (GET)
  5. 从字典中获取值 键: files
  6. 对列表中的每个项目重复 → 下载文件
